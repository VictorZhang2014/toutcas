# ToutCas

ToutCas is a French phrase called "En tout cas, ça fonctionne toujours". ToutCas is an AI assistant featured in Burn After Use, Never Data Persistence and Non-custodial Implementation. 

# Features
- ✅ Just like `ChatGPT Atlas`, an AI personal browser
- ✅ Emphasize `Burn-After-Use` and non-custodial for data persistence when interacting with the AI assistant
- ✅ First version finished in `3 Days`, continually improvement
- ✅ Support macOS, Windows, Linux, the version of iOS and Android is under development
- ✅ Inspired by this dissertation, referring to arxiv.org/xx.xxx
